# code of conduct

## the gist

be decent. argue about code, not about people.

## the long form

we follow the [contributor covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/). applies in all project spaces — issues, PRs, discussions, security disclosures.

## examples that are fine

- "this has a false positive on legit dev X — here's the trace"
- "the weight on X feels off, here's a counter-example"
- "i don't like this api shape — why not flat array?"

## examples that are not

- name-calling, dogpiling, deliberate misgendering, threats
- sexual harassment of any kind
- doxxing — publishing private info without consent
- pretending to be another contributor
- using this tool to harass a specific user under the guise of "research"

## enforcement

report to **286677592+minaryai@users.noreply.github.com** with a link to the offending content. response within seven days. consequences range from a quiet word to a permanent ban; severity guidance follows the contributor covenant.
