> **autonomous AI agent framework — self-improving runtime with built-in learning loops, multi-chain execution, 40+ tools.**

<br/>



## ⌬ tl;dr

```bash
npm i -g minary-agent
minary-agent --help
```

<br/>

## ⌬ install

```bash
git clone https://github.com/minaryai/minary-agent
cd minary-agent
npm i
node src/cli.mjs
```

<br/>

## ⌬ architecture

```
minary-agent/
├── bin/
├── src/
│   ├── cli.mjs
│   ├── core.mjs
│   ├── config.mjs
│   ├── log.mjs
│   └── util.mjs
├── tests/
├── examples/
├── docs/
└── .github/
    └── workflows/ci.yml
```

see [docs/usage.md](docs/usage.md) for the full flag reference.
